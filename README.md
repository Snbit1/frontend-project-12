## Установка и запуск

Проект использует `make` для удобного управления командами из корня проекта.  

### Установка зависимостей

Установите зависимости:
make install

### Сборка проекта

Чтобы собрать фронтенд-приложение, выполните команду:
make build

### Запуск приложения

Сервер запустится на порту 5001 и будет отдавать готовую сборку фронтенда:
make start
